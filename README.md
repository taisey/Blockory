# Blockory

## ローカル環境でMySQLの起動方法
```
cd ./local/localmysql/cmd
bash run.sh
```
## ローカル環境でGoアプリの起動方法
```
cd ./backend/src
go run main.go
```
http://localhost:8080
でアクセスできる

## ローカル環境でGoアプリとMysqlの起動方法

```
cd ./local
bash runAll.sh
```
